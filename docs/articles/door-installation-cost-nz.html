Door Installation Cost NZ 2026 — Interior, Exterior and Sliding Doors

Published 23 June 2026 · NZ Tradie Tools
door installation costdoor replacement costexterior door NZcostsNZ2026

Replacing a door — whether it's a front door, interior bedroom door, or sliding cavity door — is one of the more accessible home improvement jobs in NZ. Here's what door installation costs in NZ in 2026, what affects the price, and when you need a professional.

All prices GST inclusive, supply and install. Auckland 10–20% above. Prices vary significantly by door specification, material (timber vs aluminium vs uPVC), and whether the frame needs replacement.

Interior Door Installation

Hollow-Core vs Solid-Core

Hollow-core doors ($200–$500 supply): Light, inexpensive, standard for interior bedrooms and wardrobes. Poor sound insulation. Easy to install.

Solid-core doors ($300–$900 supply): Much better sound insulation — worthwhile for bathrooms, home offices, or any room requiring privacy. Heavier — need good hinges and a properly plumb frame.

Cavity Sliders

Cavity sliders disappear into the wall when open — excellent for space-saving where a swing door is impractical. Key considerations: - Existing cavity: Retrofitting a cavity slider into an existing wall requires opening the wall to install the pocket frame — a builder's job, not just a door supplier's. - New build or renovation: Install cavity slider frames before GIB is hung — far less disruptive. - Fire ratings: Cavity sliders between garage and living spaces may need to meet FRR (fire resistance ratings) under Building Code clause F7.

DIY Interior Doors

Hanging an interior door on an existing, plumb frame is within reach for a competent DIYer with the right tools. Key requirements: the frame must be square and plumb, the door must be trimmed accurately, and hinge positions must align. If the frame is out of square (common in older NZ homes), a carpenter will produce a better result.

Exterior Door Installation

Why Exterior Doors Cost More

Exterior doors must: - Provide weathertightness (NZ Building Code clause E2) - Meet thermal performance requirements (clause H1) - Provide security - Withstand NZ wind loads

These requirements push quality exterior doors toward aluminium-clad timber or aluminium construction with proper sealing systems. A cheap exterior door that leaks at the frame is a weathertightness problem.

Pre-Hung vs Slab Door

Pre-hung doors come with the frame attached and are ready to install into the rough opening. Best for replacements where the existing frame is being removed.

Slab doors (door only) fit into an existing frame — cheaper but require the existing frame to be in good condition, plumb, and square.

Front Door Security

For exterior doors, consider: - Multi-point locking (engages at top, middle, and bottom of the door — significantly more secure than a single-point deadbolt) - Solid core or steel door construction - Quality hinge specification (3 hinges minimum for a solid exterior door)

Sliding and Bifold Glass Doors

Aluminium Joinery

Exterior sliding and bifold doors in NZ are predominantly aluminium joinery — powder-coated or anodised. Key brands: Altherm, Vantage, First Windows and Doors, APL.

What affects cost: - Size (width × height) - Glass specification (double-glazed is standard; thermally broken frames add $800–$2,000 but dramatically improve thermal performance) - Number of panels (more panels = more cost but more opening) - Hardware quality

Weathertightness: Sliding door junctions with adjacent cladding are a common weathertightness failure point in NZ. Ensure your installer flashes the head and jamb correctly and the sill has a proper drainage detail.

Building Consent for Exterior Doors

  • Like-for-like replacement (same size, same position): Generally exempt under Schedule 1 of the Building Act.
  • New opening in an existing wall: Building consent almost always required (structural alteration + weathertightness).
  • Enlarging an existing opening: Building consent required.

Do You Need a Builder?

  • Interior door on existing frame: Handyman or carpenter, no consent needed.
  • New cavity slider (opening wall): Builder — structural work.
  • Exterior door replacement (same opening): Competent carpenter. No consent generally required.
  • New exterior opening: Builder + building consent.
  • Sliding glass / bifold exterior doors: Joinery company installs the unit; a builder may be needed if the opening is being created or altered.

Frequently Asked Questions

How much does it cost to install a door in NZ? Interior door (labour only, existing frame): $250–$600. Interior door supply + install: $350–$1,200. Exterior door supply + install: $1,200–$3,500 for a standard pre-hung door; $2,500–$7,000 for quality solid timber. Sliding glass exterior doors: $2,000–$5,000 installed.

Do I need a builder to replace a door in NZ? For an interior door replacement on an existing frame: a competent carpenter or handyman is fine. For any exterior door work involving weathertightness, or any work requiring structural alteration (new opening, enlarged opening), use a Licensed Building Practitioner (LBP). New openings in exterior walls require building consent.

Can I replace an exterior door myself in NZ? If you're replacing a like-for-like exterior door in an existing pre-hung frame, a confident DIYer can do this — but weathertightness is critical. A poorly installed exterior door that leaks will cause significant damage. For anything beyond a direct replacement, use a professional.

How long does it take to install a door? Interior door replacement on existing frame: 1–2 hours. New interior door with frame: 2–4 hours. Exterior door replacement: 3–6 hours. Sliding glass door installation: 4–8 hours depending on complexity.

What is the best exterior door brand in NZ? Altherm, Vantage, and APL are the main NZ aluminium joinery brands. For solid timber exterior doors: Stressline, Corinthian, and various custom joinery makers. For thermally-broken performance frames (worth paying for in cold climates): look for Thermally Broken certification on aluminium joinery.
