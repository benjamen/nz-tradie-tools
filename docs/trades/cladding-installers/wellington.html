How to Choose a Cladding Installer in Wellington

  • LBP licence
  • Compliance with NZ Building Code E2 (External Moisture)
  • Manufacturer's approved installation method
  • Flashing and cavity detail specification
  • 20-year manufacturer warranty on products

Red Flags to Watch Out For

  • No LBP licence
  • Won't reference E2 compliance
  • Unknown cladding brand without warranty
  • No cavity system on weathertight cladding
