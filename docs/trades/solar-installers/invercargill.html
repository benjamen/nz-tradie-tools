How to Choose a Solar Installer in Invercargill

  • REW registration
  • Solar retailer certification (SEANZ member)
  • Shading analysis and generation estimate
  • Grid connection documentation
  • 25-year panel warranty + 10-year inverter warranty

Red Flags to Watch Out For

  • No shading analysis provided
  • Generation guarantee not offered in writing
  • Not SEANZ registered
  • Won't handle grid connection documentation
